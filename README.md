# Feign Api Demo Application

### Running the Project

1. With spring boot: 
    `mvn spring-boot:run`
  
2.  Run jar file 
    ` mvn clean package`   cd target  `java -jar feign-app.jar`

3. or run directly  `FeignApiApplication`



#### H2 database
